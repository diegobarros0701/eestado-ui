# @eestado-ui
Componentes de UI do e-Estado

## Instalação

```bash
npm install @eestado-ui
```
ou
```bash
yarn add @eestado-ui
```

## Componentes

### Alerta

```javascript
import { Alerta } from '@eestado-ui';
```

### BarraContexto

```javascript
import { BarraContexto } from '@eestado-ui';
```

### Formulario

```javascript
import { Formulario } from '@eestado-ui';
```

### PreLoader
